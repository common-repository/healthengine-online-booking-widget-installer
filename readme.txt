HealthEngine plugin for online booking system

== Installation ==
Using Wordpress Library

1. Search for "HealthEngine" plugin on Wordpress admin
2. Customise "Practice Id" and "Book Now" button on widget admin page
3. Save the setting and refresh your homepage
4. You are good to go. Bon voyage ...

Manual Installation
1. Go to Wordpress plugin admin
2. Select Add Plugin and Upload Plugin
3. Select compressed HealthEngine plugin
4. Install the plugin
5. You are good to go. Bon voyage ...
